# 人体姿态识别与矢量动画还原系统
## 项目简介
本项目实现摄像头实时画面采集、人体姿态关键点识别、动作数据导出以及网页端矢量骨架动画还原的完整功能流程。依托OpenCV与MediaPipe提取人体33个全身关节关键点信息，将动作数据保存为JSON格式，最终通过HTML+SVG技术在浏览器中还原动态骨架动画，全程支持本地直接运行。

```
├── data/                      # 数据存储目录
│   ├── motion_data1.json      # 姿态识别生成的关键点数据

├── 演示视频.mp4               # 项目整体演示视频

├── requirements.txt           # Python依赖配置文件
└── vector_animation.html      # 前端骨架动画页面
```

## 技术栈
- 采集与识别端：Python、OpenCV、MediaPipe、NumPy
- 前端动画端：HTML5、JavaScript、SVG
- 数据存储：JSON
- 运行环境：Python 3.7及以上、主流浏览器（Chrome、Edge、Firefox等）

## 环境配置
1. 本地安装 **Python 3.7+** 版本
2. 打开命令行，进入项目根目录，执行以下命令安装依赖：
```bash
pip install -r requirements.txt
```

`requirements.txt` 内容：
```txt
opencv-python
mediapipe
numpy
```

## 使用方法
### 1. 摄像头实时采集与姿态识别
```bash
python code/camera_test.py
```
- 程序自动调用本地摄像头，实时识别人体姿态并绘制骨架
- 按键操作：
  - `s`：保存当前画面截图
  - `q`：退出程序并生成动作数据文件

### 2. 本地视频姿态识别
```bash
python code/capture_to_vector.py
```
- 读取`data`目录下的测试视频，逐帧提取人体关键点
- 识别完成后自动生成`motion_data1.json`数据文件

### 3. 网页播放骨架动画
1. 直接双击打开 `vector_animation.html`，无需搭建服务器
2. 在页面内上传生成的 `motion_data1.json` 文件
3. 页面自动解析数据，播放人体骨架动态动画

## 功能特点
1. 双采集模式：支持实时摄像头采集、本地视频文件解析
2. 精准姿态识别：可检测人体33个全身关键点，包含坐标与可见度信息
3. 本地免服务：采用文件上传方式读取数据，规避本地跨域限制
4. 操作简单：纯可视化界面+简易按键，上手难度低

## 注意事项
1. 运行摄像头程序前，需允许系统调用摄像头权限
2. 视频文件路径需与代码配置保持一致，否则需手动修改对应路径
3. 动画仅支持本项目导出的JSON数据，其他格式文件无法正常解析
4. 若出现画面卡顿，可降低摄像头分辨率或减少数据采样帧率
